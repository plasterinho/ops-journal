# automation glue (non-magic)

Utility scripts and tools for testing, validation, and automation.

## Files

### `__init__.py`

Marks this directory as a Python package, enabling module detection and imports from the scripts directory. This allows other Python modules to import utilities and functions from scripts.

### `test_reality.py`

Temporary development script for testing the reality checks engine. Used to validate and debug the reality module functionality during development before integration into the main system.

### `validate-weeks.js`

JavaScript utility for validating week-based data and journal entries.
